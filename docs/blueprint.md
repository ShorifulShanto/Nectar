# **App Name**: Olipop Odyssey

## Core Features:

- Cinematic Scroll Animation: Implement a scroll-controlled WebP image sequence animation for an immersive and cinematic display of drink variants.
- Dynamic Product Display: Enable switching between multiple drink variants, dynamically updating visual content, text overlays, and interactive elements.
- Firebase Data Integration: Utilize Firebase Firestore to efficiently store and manage all drink variant metadata, image sequence paths, and textual content.
- Optimized Canvas Rendering: Leverage HTML5 Canvas for high-performance and smooth rendering of WebP animation sequences, ensuring a fluid user experience.
- Intuitive Navigation System: Provide a vertical navigation for seamless flavor selection and a sticky top navigation bar for primary site sections and branding.
- Progressive Asset Loading: Implement intelligent initial preloading and lazy loading mechanisms for image sequences to ensure rapid display and smooth variant transitions.
- AI-Enhanced Product Storytelling: An AI tool to generate captivating, unique short narrative descriptions for each drink flavor, enriching content dynamically.

## Style Guidelines:

- Primary Color: A vibrant electric purple (#663BFF) chosen to represent the modern, functional, and energetic brand ethos, providing a striking contrast against the dark background. (HSL(255, 75%, 55%))
- Background Color: Predominantly pure black (#000000) as per the dark-mode theme, establishing a premium, cinematic backdrop for the dynamic content. White (#FFFFFF) will be used for essential text elements.
- Accent Color: A crisp sky blue (#A0DAFF), providing a refreshing visual highlight for interactive elements and calls to action, complementing the primary purple. (HSL(225, 80%, 75%))
- Dynamic Flavor Colors: Integrate the specified flavor-specific colors (Green, Light Green, Pink, Deep Red, Orange, Yellow, Purple) to visually differentiate and highlight selected drink variants, reinforcing brand identity.
- Headline Font: 'Space Grotesk' (sans-serif) for large, bold, and impactful headings like drink names, contributing to a modern, slightly tech-inspired, and premium aesthetic.
- Body Font: 'Inter' (sans-serif) for all descriptive text, subtitles, navigation links, and UI labels, ensuring high legibility and a clean, objective presentation across the interface.
- Employ a consistent set of minimal, monochrome outline icons, such as clean arrows for navigation and brand-standard logos for social media, maintaining the sleek, uncluttered dark UI.
- Implement a sophisticated, full-screen, grid-based layout that prioritizes content visibility and interactivity. The hero section follows a defined left-text, center-animation, right-navigation structure, complemented by a sticky top navbar and consistent section styling for clarity and user flow.
- Feature highly optimized, scroll-driven cinematic animations with requestAnimationFrame for fluid frame progression. Implement smooth, debounced transitions like subtle fades and scaling for content switching and interactive elements, ensuring a high-performance and seamless user experience.
